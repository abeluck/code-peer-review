Hall of Fame

Project Contributors are individuals who have used the Code Peer Review system for its exact intention: finding a project and reporting a bug, identifying an issue, or contributing a patch. We want to connect people with projects to increase the quality and security of open source projects with the least friction possible.

Hackers are individuals who have found a major security issue in a project or the Code Peer Review system itself. You may earn Project Contributor or Contributor status also - but we want to recognize the skill and dedication it takes to find and identify security issues.

Contributors have donated their time and code to the Code Peer Review platform to improve it for all of its users.
